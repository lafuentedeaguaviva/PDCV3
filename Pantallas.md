DESCRIPCIÓN DE PANTALLAS DEL SISTEMA
Sistema de Planificación Curricular
DESCRIPCIÓN GENERAL DEL SISTEMA

El Sistema de Planificación Curricular es una plataforma educativa diseñada para facilitar la organización, creación y seguimiento de planes de desarrollo curricular en unidades educativas. Permite a docentes crear PDC, planificar sus clases por semanas, organizar contenidos, definir momentos pedagógicos, asignar recursos y establecer criterios de evaluación. Los directores pueden supervisar las planificaciones de los docentes de su unidad, mientras que los administradores gestionan la estructura académica y los contenidos base del sistema. La plataforma opera bajo un modelo de suscripción por créditos, donde los usuarios adquieren planes que les permiten acceder a funcionalidades según sus necesidades.
PANTALLAS DEL SISTEMA
1. LANDING PAGE (PÁGINA DE INICIO)

Descripción:
Página principal del sistema, diseñada para atraer y convertir visitantes en usuarios registrados. Presenta una visión general del producto, sus beneficios y los planes disponibles.

Elementos principales:

    Encabezado: Logo del sistema, menú de navegación (Inicio, Características, Planes, Contacto), botones de "Iniciar Sesión" y "Registrarse".

    Sección Hero: Mensaje principal impactante como "Planifica tus clases de manera fácil y eficiente", con una imagen representativa y llamado a la acción "Comenzar ahora".

    Sección de Características: Grid con íconos y descripciones de las funcionalidades clave: Gestión de contenidos, Planificación semanal, Momentos pedagógicos, Recursos educativos, Evaluación, Adaptaciones curriculares.

    Sección de Planes: Presentación de los diferentes planes de suscripción con sus características y precios, destacando el plan más popular. Cada plan muestra cantidad de créditos, funcionalidades incluidas y límites de uso.

    Sección de Testimonios: Opiniones de usuarios reales sobre su experiencia con el sistema.

    Sección de Preguntas Frecuentes: Acordeón con respuestas a las dudas más comunes.

    Pie de Página: Enlaces legales, contacto, redes sociales y copyright.

Propósito: Atraer nuevos usuarios, informar sobre el producto y convertir visitantes en clientes mediante la compra de planes.
2. PANTALLA DE COMPRA DE PLANES

Descripción:
Pantalla donde los usuarios nuevos o existentes pueden adquirir planes de créditos para acceder a las funcionalidades del sistema.

Elementos principales:

    Encabezado: Título "Elige el plan que mejor se adapte a tus necesidades".

    Selector de Planes: Tres opciones de planes (Básico, Profesional, Institucional) mostradas en tarjetas comparativas.

        Plan Básico: Precio mensual, créditos incluidos, funcionalidades limitadas (hasta 3 áreas, contenidos básicos).

        Plan Profesional: Precio mensual, créditos incluidos, funcionalidades completas (áreas ilimitadas, contenidos, recursos, evaluaciones).

        Plan Institucional: Precio personalizado, créditos para equipos, funcionalidades completas con gestión de múltiples usuarios.

    Tabla Comparativa: Características detalladas lado a lado para facilitar la comparación.

    Calculadora de Créditos: Permite al usuario simular cuántos créditos necesitará según sus actividades.

    Formulario de Pago: Campos para datos de tarjeta, PayPal u otros métodos de pago integrados.

    Resumen de Compra: Detalle del plan seleccionado, costo total y créditos a recibir.

    Botón de Confirmación: "Comprar ahora" o "Adquirir plan".

Propósito: Facilitar la adquisición de planes de créditos para que los usuarios puedan acceder a las funcionalidades del sistema según sus necesidades.
3. PANTALLA DE LOGIN (INICIO DE SESIÓN)

Descripción:
Pantalla de acceso para usuarios registrados, donde ingresan sus credenciales para acceder al sistema.

Elementos principales:

    Logo del Sistema: Identificador visual en la parte superior.

    Título: "Iniciar Sesión en tu Cuenta".

    Campo de Email: Entrada de texto para el correo electrónico.

    Campo de Contraseña: Entrada de texto tipo password.

    Enlace "Olvidé mi contraseña": Redirige a la pantalla de recuperación.

    Botón de "Iniciar Sesión": Acción principal para autenticarse.

    Enlace a Registro: "¿No tienes cuenta? Regístrate aquí" para nuevos usuarios.

    Opción de Recordar Sesión: Checkbox para mantener la sesión activa.

    Mensajes de Error: Retroalimentación en caso de credenciales incorrectas o cuenta no verificada.

Propósito: Autenticar usuarios y otorgar acceso seguro al sistema.
4. PANTALLA DE REGISTRO DE USUARIO

Descripción:
Pantalla para que nuevos usuarios creen una cuenta en el sistema.

Elementos principales:

    Logo del Sistema: Identificador visual en la parte superior.

    Título: "Crea tu Cuenta Gratuita".

    Campo de Nombres: Entrada de texto para nombres.

    Campo de Apellidos: Entrada de texto para apellidos.

    Campo de Email: Entrada de texto para correo electrónico.

    Campo de Contraseña: Entrada de texto tipo password.

    Campo de Confirmar Contraseña: Entrada de texto tipo password.

    Checkbox de Términos y Condiciones: Aceptación de políticas de uso.

    Botón de "Registrarme": Acción para crear la cuenta.

    Enlace a Login: "¿Ya tienes cuenta? Inicia sesión aquí".

    Mensajes de Validación: Retroalimentación en tiempo real sobre la validez de los campos.

Propósito: Permitir el registro de nuevos usuarios con creación automática de perfil básico y asignación de rol de Profesor.
5. PANTALLA DE PERFIL DE USUARIO

Descripción:
Pantalla donde el usuario completa y edita su información personal después del registro.

Elementos principales:

    Título: "Mi Perfil".

    Foto de Perfil: Imagen actual con opción de cambiar o eliminar.

    Campo de Título Profesional: Selector desplegable con opciones (Prof., Lic., Ing., etc.).

    Campo de Nombres: Entrada de texto editable.

    Campo de Apellidos: Entrada de texto editable.

    Campo de Email: Mostrado pero no editable (verificado por autenticación).

    Campo de Celular: Entrada de texto para número de contacto.

    Barra de Progreso: Indicador de porcentaje de completitud del perfil.

    Lista de Roles Asignados: Visualización de roles del usuario (Profesor, Director, Administrador).

    Botón de "Guardar Cambios": Acción para actualizar la información.

    Botón de "Cambiar Contraseña": Redirige a pantalla de actualización de credenciales.

Propósito: Permitir a los usuarios mantener actualizada su información personal y completar su perfil.
6. DASHBOARD PRINCIPAL

Descripción:
Pantalla de inicio después del login, personalizada según el rol del usuario (Profesor, Director o Administrador).

Elementos principales:

    Barra Superior: Logo, buscador global, notificaciones, menú de usuario.

    Menú Lateral: Navegación principal entre módulos del sistema.

    Tarjetas de Resumen: Indicadores clave según el rol.

        Profesor: Mis áreas, Planificaciones recientes, Semanas pendientes, Contenidos propios.

        Director: Docentes activos, Áreas supervisadas, Semanas planificadas, Alertas.

        Administrador: Usuarios totales, Unidades educativas, Contenidos base, Créditos emitidos.

    Lista de Áreas de Trabajo: Acceso rápido a las áreas del profesor o de los docentes supervisados.

    Calendario Resumido: Próximas semanas con planificaciones pendientes.

    Actividad Reciente: Últimas planificaciones realizadas o modificadas.

    Accesos Directos: Botones para acciones comunes (Nueva planificación, Nuevo contenido, etc.).

    Notificaciones: Alertas sobre planificaciones pendientes o contenidos compartidos.

Propósito: Proporcionar una visión general del estado del sistema y acceso rápido a las funcionalidades principales.
7. PANTALLA DE UNIDADES EDUCATIVAS

Descripción:
Pantalla para gestionar la estructura geográfica y administrativa de las unidades educativas (solo para administradores).

Elementos principales:

    Selector de Nivel Geográfico: Pestañas o breadcrumbs para navegar entre Departamentos, Distritos y Unidades.

    Lista de Departamentos: Tabla con nombre, fecha de creación y acciones (editar, eliminar).

    Lista de Distritos: Tabla con nombre, departamento asociado y acciones.

    Lista de Unidades Educativas: Tabla con nombre, código, distrito, director asignado y acciones.

    Botón de "Agregar Nuevo": Para crear departamentos, distritos o unidades según el nivel seleccionado.

    Formulario Modal: Ventana emergente para ingresar datos al crear o editar.

    Asignación de Director: Selector de usuarios con rol de Director para asignar a una unidad.

    Buscador y Filtros: Para localizar rápidamente elementos en las listas.

    Exportar Datos: Opción para descargar listados en formato CSV o PDF.

Propósito: Administrar la estructura geográfica que soporta todo el sistema educativo.
8. PANTALLA DE ESTRUCTURA ACADÉMICA

Descripción:
Pantalla para gestionar los catálogos académicos: niveles, grados, áreas de conocimiento, turnos y paralelos.

Elementos principales:

    Pestañas de Navegación: Separadores para cada catálogo (Niveles, Grados, Áreas, Turnos, Paralelos).

    Lista de Niveles: Tabla con nombre, objetivo holístico y acciones.

    Lista de Grados: Tabla con nombre, nivel asociado y acciones.

    Lista de Áreas de Conocimiento: Tabla con nombre, descripción, grado asociado y acciones.

    Lista de Turnos: Tabla simple con nombre y acciones.

    Lista de Paralelos: Tabla simple con nombre y acciones.

    Botón de "Agregar Nuevo": Para crear elementos en el catálogo activo.

    Relaciones Visuales: Indicadores de jerarquía (ej. área pertenece a grado, grado pertenece a nivel).

    Buscador: Para filtrar elementos en listas largas.

    Mensajes de Advertencia: Alertas al intentar eliminar elementos con dependencias.

Propósito: Mantener actualizada la estructura académica que define las posibilidades de planificación.
9. PANTALLA DE CONTENIDOS BASE (ADMIN)

Descripción:
Pantalla para que administradores gestionen los contenidos base que servirán como plantillas oficiales para los docentes.

Elementos principales:

    Selector de Área: Filtro para elegir el área de conocimiento.

    Vista Jerárquica: Árbol expandible de contenidos (temas principales y subtemas).

    Lista de Contenidos: Tabla con título, área, orden y acciones.

    Botón de "Nuevo Contenido": Crear un contenido base.

    Botón de "Nuevo Subtema": Crear un subtema dentro de un tema existente.

    Formulario de Contenido: Campos para título, descripción, área, orden y padre (para subtemas).

    Controles de Ordenamiento: Flechas para reorganizar la jerarquía.

    Buscador: Para localizar contenidos por título o descripción.

    Vista Previa: Visualización de cómo se verá el contenido en la biblioteca de docentes.

Propósito: Proveer una biblioteca oficial de contenidos que los docentes puedan copiar y adaptar.
10. PANTALLA DE BIBLIOTECA DE CONTENIDOS

Descripción:
Pantalla donde los profesores pueden ver, buscar y gestionar sus contenidos, así como acceder a contenidos base y compartidos.

Elementos principales:

    Pestañas: "Mis Contenidos", "Contenidos Base", "Compartidos conmigo", "Públicos".

    Vista de Cuadrícula/Lista: Toggle para cambiar entre modos de visualización.

    Tarjetas de Contenido: Muestran título, tipo (propio/base/compartido), fecha de modificación, y acciones.

    Buscador y Filtros: Por título, área, grado, fecha.

    Acciones por Contenido: Editar, copiar, compartir, eliminar, ver detalles.

    Botón de "Nuevo Contenido": Crear contenido desde cero.

    Botón de "Copiar desde Base": Acceso rápido a contenidos base para copiar.

    Modal de Compartir: Selector de usuarios para compartir contenidos específicamente.

    Indicador de Compartición: Iconos que muestran si un contenido es público o compartido.

Propósito: Centralizar la gestión de contenidos de usuario y facilitar la reutilización mediante copias y compartición.
11. PANTALLA DE EDITOR DE CONTENIDO

Descripción:
Pantalla para crear o editar un contenido de usuario, con opciones de formato y organización jerárquica.

Elementos principales:

    Título del Contenido: Campo editable para el título.

    Descripción: Área de texto enriquecido (negrita, listas, etc.).

    Selector de Área: Asignación a área de conocimiento (si aplica).

    Orden: Número para posicionamiento dentro del nivel jerárquico.

    Selector de Padre: Para definir si es subtema de otro contenido.

    Opciones de Compartición: Checkbox para "Compartir públicamente".

    Vista Preliminar: Botón para ver cómo se verá el contenido terminado.

    Botones de Acción: Guardar, Guardar y salir, Cancelar.

    Indicador de Origen: Mensaje informativo si el contenido fue copiado de base o de otro usuario.

    Historial de Versiones: Lista de cambios realizados (opcional).

Propósito: Permitir la creación y edición detallada de contenidos educativos.
12. PANTALLA DE ÁREAS DE TRABAJO

Descripción:
Pantalla para que los profesores gestionen sus áreas de trabajo, asignando paralelos y horarios.

Elementos principales:

    Lista de Áreas de Trabajo: Tarjetas o tabla mostrando cada área con sus detalles.

    Botón de "Nueva Área": Crear una nueva área de trabajo.

    Formulario de Área: Selectores para Unidad Educativa, Área de Conocimiento, Turno.

    Gestión de Paralelos: Lista de paralelos asignados con sus horarios.

    Botón de "Agregar Paralelo": Añadir un paralelo con su horario.

    Editor de Horario: Interfaz visual para definir horarios por día (lunes a viernes).

    Asignación de Contenido: Selector para elegir el contenido activo del área.

    Acciones por Área: Editar, Duplicar, Eliminar, Ver planificaciones.

    Resumen de Carga Horaria: Visualización del total de horas por semana.

Propósito: Configurar las asignaciones docentes con sus respectivos paralelos y horarios.
13. PANTALLA DE CALENDARIO/PLANIFICACIÓN

Descripción:
Vista de calendario que muestra las semanas del año con sus planificaciones, permitiendo navegar por trimestres y meses.

Elementos principales:

    Selector de Año: Control para elegir el año académico.

    Vista de Trimestres: Pestañas para cambiar entre Trimestre 1, 2 y 3.

    Calendario Mensual: Grid mostrando las semanas de cada mes.

    Tarjetas de Semana: Cada semana muestra fechas y estado (planificada, pendiente, completada).

    Indicador de Contenidos: Pequeños puntos o números que indican cuántos contenidos tiene cada semana.

    Acceso Rápido: Clic en una semana para ir a la pantalla de planificación detallada.

    Leyenda de Estados: Colores que identifican el estado de cada semana.

    Filtros: Por área de trabajo, por estado.

    Botón de "Planificar Semana": Acceso directo para semanas sin planificar.

Propósito: Visualizar de manera general la distribución de planificaciones a lo largo del año.
14. PANTALLA DE PLANIFICACIÓN SEMANAL

Descripción:
Pantalla detallada para planificar una semana específica, asignando contenidos, momentos y recursos.

Elementos principales:

    Encabezado de Semana: Información de fechas, trimestre, mes y semana.

    Selector de Área de Trabajo: Si el profesor tiene múltiples áreas, elige para cuál planificar.

    Lista de Contenidos Planificados: Contenidos asignados a la semana con su orden.

    Botón de "Agregar Contenido": Abre selector de biblioteca para elegir contenido.

    Tarjeta de Contenido: Muestra título, orden, observaciones, y opciones.

    Controles de Orden: Flechas para reorganizar contenidos.

    Sección de Momentos: Para cada contenido, lista de momentos con tipo y descripción.

    Sección de Recursos: Lista de recursos asignados a la semana.

    Campo de Observaciones: Notas generales para la semana.

    Selector de Estado: Dropdown para cambiar estado (pendiente, planificado, completado).

    Botones de Acción: Guardar, Guardar y continuar, Cancelar.

Propósito: Permitir la planificación detallada de cada semana con todos sus elementos.
15. PANTALLA DE MOMENTOS

Descripción:
Interfaz para gestionar los momentos pedagógicos de un contenido planificado en una semana.

Elementos principales:

    Encabezado: Título del contenido y semana asociada.

    Lista de Momentos: Secuencia de momentos con tipo y descripción.

    Botón de "Agregar Momento": Añadir un nuevo momento.

    Selector de Tipo: Dropdown con opciones (Práctica, Teoría, Producción, Valoración).

    Campo de Descripción: Texto explicativo del momento.

    Control de Orden: Números o flechas para reordenar.

    Acciones por Momento: Editar, Eliminar.

    Vista Preliminar: Visualización de cómo se verá la secuencia completa.

    Botones de Acción: Guardar momentos, Cancelar.

Propósito: Definir la secuencia pedagógica de cada contenido planificado.
16. PANTALLA DE RECURSOS

Descripción:
Pantalla para gestionar recursos educativos que pueden ser asignados a las planificaciones.

Elementos principales:

    Lista de Recursos: Tabla o cuadrícula con todos los recursos disponibles.

    Botón de "Nuevo Recurso": Crear un recurso.

    Formulario de Recurso: Campos para tipo, título, autor, editorial, año, ISBN, URL.

    Selector de Tipo: Dropdown con opciones (Libro, Video, Presentación, Ejercicio, etc.).

    Buscador: Por título, autor, tipo.

    Acciones por Recurso: Editar, Eliminar, Ver detalles.

    Asignación a Semanas: Indicador de en qué semanas está siendo usado.

    Vista de Detalle: Información completa del recurso.

Propósito: Centralizar la gestión de recursos educativos reutilizables.
17. PANTALLA DE CRITERIOS DE EVALUACIÓN

Descripción:
Pantalla para que los profesores definan los criterios de evaluación para cada área de trabajo.

Elementos principales:

    Selector de Área: Dropdown para elegir el área de trabajo.

    Lista de Criterios: Tabla mostrando descripción, dimensión, orden y estado.

    Botón de "Nuevo Criterio": Agregar criterio.

    Formulario de Criterio: Campos para descripción, dimensión (ser, saber, hacer), orden.

    Checkbox de Activo: Para activar/desactivar criterios sin eliminarlos.

    Controles de Orden: Flechas para reorganizar.

    Acciones por Criterio: Editar, Eliminar, Activar/Desactivar.

    Vista Previa: Cómo se verán los criterios en la planificación.

Propósito: Definir los estándares de evaluación para cada área.
18. PANTALLA DE ADAPTACIONES CURRICULARES

Descripción:
Pantalla para registrar adaptaciones curriculares específicas para un contenido en una semana determinada.

Elementos principales:

    Selector de Semana y Contenido: Identificación de qué contenido se está adaptando.

    Lista de Adaptaciones: Adaptaciones registradas para ese contenido.

    Botón de "Nueva Adaptación": Agregar adaptación.

    Campo de Tipo: Texto para tipo de adaptación (Visual, Auditiva, Motriz, Cognitiva, etc.).

    Campo de Descripción: Detalle de la adaptación.

    Campo de Recursos Adicionales: Materiales necesarios.

    Campo de Criterios Adaptados: Cómo se modifican los criterios de evaluación.

    Acciones por Adaptación: Editar, Eliminar.

    Botones de Acción: Guardar adaptaciones, Cancelar.

Propósito: Documentar las adaptaciones curriculares necesarias para atender diversidad.
19. PANTALLA DE GESTIÓN DE USUARIOS (ADMIN)

Descripción:
Pantalla para que administradores gestionen todos los usuarios del sistema, sus roles y estados.

Elementos principales:

    Lista de Usuarios: Tabla con columnas: Nombre, Email, Roles, Estado del perfil, Créditos, Acciones.

    Buscador y Filtros: Por nombre, email, rol, estado.

    Botón de "Nuevo Usuario": Crear usuario manualmente.

    Acciones por Usuario: Editar, Asignar roles, Ver detalles, Suspender, Eliminar.

    Modal de Asignación de Roles: Interfaz para agregar/quitar roles (especialmente Director).

    Asignación a Unidad: Si se asigna rol Director, selector para elegir unidad educativa.

    Reporte de Créditos: Visualización de saldo y transacciones por usuario.

    Exportar Listado: Opción para descargar listado de usuarios.

Propósito: Administración centralizada de usuarios y sus permisos.
20. PANTALLA DE REPORTES Y ESTADÍSTICAS

Descripción:
Pantalla para visualizar reportes y estadísticas del sistema, adaptada según el rol.

Elementos principales:

    Selector de Tipo de Reporte: Opciones predefinidas (Planificaciones por docente, Contenidos más usados, etc.).

    Selector de Período: Rango de fechas, trimestre o año.

    Gráficos Visuales: Barras, líneas, pasteles según el reporte.

    Tablas de Datos: Información detallada en formato tabular.

    Exportar: Botones para exportar a PDF, Excel o CSV.

    Filtros Adicionales: Por unidad educativa, área, docente.

    Resumen Ejecutivo: Indicadores clave en tarjetas superiores.

    Botón de "Generar Reporte": Acción para actualizar los datos.

Propósito: Proveer información valiosa para la toma de decisiones.
21. PANTALLA DE CRÉDITOS

Descripción:
Pantalla donde los usuarios pueden ver su saldo de créditos, historial de transacciones y adquirir más.

Elementos principales:

    Tarjeta de Saldo Actual: Muestra créditos disponibles con diseño destacado.

    Historial de Transacciones: Tabla con fecha, cantidad, motivo, referencia.

    Gráfico de Consumo: Visualización de uso de créditos en el tiempo.

    Botón de "Comprar Créditos": Redirige a pantalla de compra de planes.

    Detalle de Consumo: Desglose de créditos gastados por módulo o acción.

    Alertas: Mensajes cuando el saldo es bajo.

    Exportar Historial: Opción para descargar movimientos.

Propósito: Gestionar el sistema de créditos y permitir a los usuarios controlar su consumo.
